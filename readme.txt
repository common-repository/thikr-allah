== Description ==
 When we are working in the admin dashboard we forgot tikr allah , this plugin in cha allah will help you remember the tikr with short thikr statments,(This plugin based in Hello Dolly idea)


== Installation ==
1. Download file 
2. Upload and extract in plugin folder, or upload by add new plugin at your WordPress Dashboard
3. Activate the plugin through the \'Plugins\' menu in WordPress
4. Enjoy it.
